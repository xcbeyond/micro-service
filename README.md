# 微服务系列知识(micro-service)

目前微服务的开发框架比较流行的有SpringBoot、SpringCloud。
* SpringBoot：
  旨在简化创建产品级的Spring应用和服务，简化了配置文件，使用嵌入式web服务器，含有诸多开箱即用微服务功能，可以和spring cloud联合部署。
* SpringCloud：
  微服务工具包，为开发者提供了在分布式系统的配置管理、服务发现、断路器、智能路由、微代理、控制总线等开发工具包。
